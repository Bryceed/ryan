How to use?

Hosted by us

  1. Invite the RYAN BOT to your server by clicking here.
  2. Give me a role that has permission to manage messages.

Host your own (for developers)

  1. Clone the repository.
  2. Install Node.js under the version LTS 16.6.0 or higher
  3. Also, install any code editor you want (I recommend Visual Studio Code).
  4. Access the root folder of this project with the terminal and install its dependencies with npm install.
  5. Copy the .env.example file and rename it to .env and fill in the values.
  6. Do the same with development file example, to enable the usage under development mode. If you don't have an second bot to test it, use the same values from the step above
  7. Run the bot in development mode with npm run dev or in production mode (host) with npm start.
  8. Host the bot somewhere to keep it "online 24/7" (depending where you are hosting the code, how much you wanna pay to keep it, how many servers you want to serve, periferical stuff like RAM, CPU, etc.)
